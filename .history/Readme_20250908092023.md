# Compliance Frontend

Este repositório contém a parte frontend do sistema Compliance, desenvolvido em Angular.

## Estrutura dos Repositórios

O sistema Compliance foi dividido em dois repositórios:
- **compliance-frontend**: Este repositório (interface de usuário em Angular)
- **compliance-backend**: Repositório separado com a API backend (Spring Boot)

## Configuração do Ambiente

Para executar o sistema completo com Docker Compose, siga estas etapas:

1. Clone ambos os repositórios na mesma pasta pai:
   ```bash
   mkdir compliance-sistema
   cd compliance-sistema
   git clone https://github.com/Compliance-STJ/compliance-frontend.git
   git clone https://github.com/Compliance-STJ/compliance-backend.git
   ```

2. Navegue até a pasta do frontend:
   ```bash
   cd compliance-frontend
   ```

3. Execute o Docker Compose para iniciar todos os serviços:
   ```bash
   docker-compose up
   ```

## Acessar os Serviços:

- **Frontend (Angular)**: http://localhost:4200
- **Backend (Spring Boot)**: http://localhost:8080
- **pgAdmin**: http://localhost:5050 (use o email: admin@example.com e senha: admin)

## Desenvolvimento

Para desenvolvimento sem Docker, você pode executar o frontend independentemente:

```bash
cd compliance-web
npm install
ng serve
```
